# portfolio
## Requirement
- Materialize
- Jinja2

## Commands
- activate virtualenv
```console
$ source venv/bin/activate
```
- generate HTML file (index.html)
```console
$ python3 builder.py
```
- deactivate virtualenv
```console
$ deactivate
```
